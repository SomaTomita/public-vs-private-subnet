    @app.route("/fetch")
def fetch():
    """SSRF VULNERABLE ENDPOINT - FOR EDUCATION ONLY"""
    url = request.args.get("url", "")
    if not url:
        return "Usage: /fetch?url=http://example.com", 400
    try:
        resp = requests.get(url, timeout=5)
        return resp.text
    except Exception as e:
        return str(e), 500

    IMDSv1 が有効化されています

    http_tokens = "optional" により、SSRF だけで EC2 メタデータからクレデンシャルが盗まれます。

    terraform/ec2.tf:64-67

    過剰な IAM Role Policy

    • • AmazonS3ReadOnlyAccess
    • • AmazonEC2ReadOnlyAccess
    • • AmazonRDSReadOnlyAccess
    • • AmazonSSMManagedInstanceCore

    盗まれたクレデンシャルは VPC 境界を超えて AWS API 層で動作するため、Private Subnet では防げません。
